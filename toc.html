Chapter 1: Introduction

1.1 Privacy in Computing

Privacy refers to keeping sensitive information safe from unauthorized access. Data such as financial transactions, personal records, and medical information must be secured.

1.2 Role of TOC in Secure Systems

TOC explores what can be computed, efficiency, and limits. Its role in privacy includes:

  • Automata → modeling and enforcing rules in protocols.
  • Algorithms → encryption, hashing.
  • Complexity → ensuring breaking security is infeasible.

1.3 Encryption as a TOC Application

Encryption schemes are algorithms running on Turing Machines. Security relies on computational complexity (e.g., RSA based on integer factorization).

Chapter 3: Algorithms in Encryption and Data Security

3.1 Definition

Algorithms are step-by-step procedures, crucial for encryption, hashing, digital signatures.

3.2 Encryption Algorithms

Symmetric: AES, DES. Same key used for encryption and decryption.

Asymmetric: RSA, ECC. Public/private key pairs.

3.3 Hashing Algorithms

SHA-256, MD5. One-way functions ensuring integrity and authentication.

3.4 Privacy-Preserving Algorithms

  • Homomorphic Encryption → compute on encrypted data.
  • Zero-Knowledge Proofs → prove knowledge without revealing.
  • Differential Privacy → add noise to datasets.

3.5 Applications

AlgorithmRoleApplications
SymmetricFast bulk encryptionBanking, Wi-Fi
AsymmetricSecure key exchangeEmail, SSL
HashingIntegrity, authenticationBlockchain, passwords
Privacy-PreservingCompute securelyCloud, voting

Chapter 4: Computational Complexity and Security

4.1 Definition

Study of resources (time, memory) needed to solve problems.

4.2 Complexity Classes

P, NP, NP-Complete, NP-Hard. P vs NP remains unsolved.

4.3 Hard Problems

Integer Factorization → RSA. Discrete Logarithm → ECC. Lattice Problems → Post-Quantum.

4.4 Security Principles

Brute force infeasible. AES-256 safe. Quantum computing threatens RSA/ECC (Shor’s algorithm).

4.5 Applications

ProblemUseStrength
FactorizationRSAStrong until quantum
Discrete LogECCEfficient
LatticePost-quantumResistant to quantum
